# 修改说明

## 1. sketch.js
**首选入口文件**  
- 包含 p5.js 的 `setup()` 和 `draw()` 循环。  
- 会调用 `controller.update()`、`display.show()` 等。  
- **修改提示**：如果要改变游戏的整体流程或画面更新逻辑，请先从这里开始。

---

## 2. controller.js
**游戏逻辑核心（状态机）**  
- 管理 `gameState`（`IDLE`、`MIX`、`GUESS`、`REVEAL`、`SCORE`）。  
- 处理键盘/硬件输入，决定何时切换阶段。  
- **修改提示**：要实现“齿轮作为 trigger、分数计算、回合切换”，就在这里添加逻辑。

---

## 3. display.js
**绘制界面**  
- 控制画面绘制：色轮、玩家指针、mask、HUD。  
- **修改提示**：如果要修改 UI 样式（色盘显示、遮罩、动画），就在这里动手。

---

## 4. player.js
**玩家对象**  
- 定义玩家属性（位置、得分）。  
- **修改提示**：要加“红蓝两人独立指针”的逻辑，从这里改起。

---

## 5. animation.js
**动画效果**  
- 管理 `reveal`、`collision` 等特殊效果。  
- **修改提示**：如果要实现“radial wipe 显示色盘”，就在这里扩展。

---

## 6. `index.html` + `style.css`
**外围 UI**  
- 一般不用大改。  
- **修改提示**：仅在需要修改网页标题、样式或添加说明文字时才编辑。

---

## 一般修改顺序（从最主要的界面入手）
1. **`sketch.js`** → 确认主循环调用和画面刷新逻辑  
2. **`controller.js`** → 修改状态机，加上 `MIX`/`GUESS`/`REVEAL` 的规则  
3. **`display.js`** → 绘制色盘、遮罩和玩家指针  
4. **`player.js`** → 确保两位玩家能独立控制位置和得分  
5. **`animation.js`** → 需要动画效果时再扩展
